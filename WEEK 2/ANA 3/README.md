# ANA-3 : Cross-Dataset Comparison

This analysis compares findings from the APMS 2014 dataset and the CSEW dataset.

The analysis focuses on:
- Mental health treatment prevalence
- Domestic abuse prevalence
- Gender-based comparison
- Trend analysis
- Social impact indicators

Datasets Used:
1. APMS 2014 Dataset (Table 3.16)
2. CSEW Dataset (Table S43c)

Tools Used:
- Python
- Pandas
- Matplotlib
- Google Colab
